One-Player Game Rules

1. Press any key to start game
2. Simon will give the first signal. Repeat the signal by pressing the same color lens
3. Level up, Simon will show the second signal. You need to repeat your first signal plus the second signal
4. Level up, Simon will show the thrid signal. You need to repeat the first two signals, plus the third one
5. If you fail to repeat, you lose
6. Press any key to restart the game
